# Pre-deploy live-test checklist (v1.2.0)

**Sprint 16G commitment**: Vóór elke FTP-deploy doet Claude (orchestrator) zelf een browser-MCP playwright-test op localhost. Geen "live verify door user" meer als enige test.

## Test-flow (mandatory)

### 1. Build to /tmp + start vite preview
```bash
cd /Users/richardtheuws/Documents/games/cosmos-cosmic-adventure-2026
rm -rf /tmp/cosmos-v120-deploy
VITE_BASE=/games/cosmos-2026/ npx vite build --outDir /tmp/cosmos-v120-deploy
DIST_DIR=/tmp/cosmos-v120-deploy node scripts/postbuild-rewrite-paths.mjs
DIST_DIR=/tmp/cosmos-v120-deploy node scripts/postbuild-copy-public.mjs
npx vite preview --outDir /tmp/cosmos-v120-deploy --port 4173 --host 0.0.0.0 &
```
(Of staging via host.docker.internal of een tijdelijke FTP-staging-folder.)

### 2. Browser-MCP playwright (iPhone 14 Pro emulation)
```
mcp__MCP_DOCKER__browser_navigate http://host.docker.internal:4173/games/cosmos-2026/play/
```

### 3. Visual checkpoints (screenshot om de 5s, 0-30s)

**0s — pre-touch**:
- [ ] Boot-overlay zichtbaar
- [ ] "tik om te ontwaken" microcopy zichtbaar
- [ ] Geen console errors

**3s — post-first-touch (auto-tap via browser_click center)**:
- [ ] Audio context unlocked
- [ ] Music begint (controleer audioBridge.musicCurrentTime > 0 via browser_evaluate)
- [ ] Nevel-portal expanding zichtbaar
- [ ] Cosmo verschijnt (niet de oude PIL-blob, nieuwe LoRA-Cosmo of geverifieerde 2D-fallback)

**5s — Cosmo arrived**:
- [ ] Cosmo center, DNA-correct zichtbaar (chameleon eyes proper, suction-cup discs, antenne+flower)
- [ ] Wave-uncanny anim playing (te-traag, eyes-locked)
- [ ] Gibberish-coo audio plays (NIET NL voice-over)

**10s — walking + first hint**:
- [ ] Cosmo loopt rechts
- [ ] HintGlyph "veeg omhoog" zichtbaar bij eerste obstakel
- [ ] Eerste obstakel zichtbaar (organic-flesh-trampoline of variant — random)

**15-30s — gameplay**:
- [ ] Mouth-pillar opent-sluit op-beat (NIET 4 monden tegelijk)
- [ ] Trampoline frequent zichtbaar (40% van low-spawns)
- [ ] Geen 3× zelfde obstakel achter elkaar
- [ ] Spacing 2.4-3.8s
- [ ] Music continu (geen drop-out)
- [ ] Secret-crystal NIET zichtbaar (tenzij DeepTripMode actief)

### 4. Audio verification
```javascript
// Via browser_evaluate
const cosmos = window.cosmos;
const audioEl = cosmos.audioBridge?.source?.audioEl;
return {
  paused: audioEl?.paused,
  currentTime: audioEl?.currentTime,
  duration: audioEl?.duration,
  loop: audioEl?.loop,
  src: audioEl?.src,
};
```
- [ ] paused === false
- [ ] currentTime > 0
- [ ] loop === true
- [ ] src ends in `.mp3`

### 5. Console assertions
```javascript
// Via browser_console_messages level=error all=false
```
- [ ] 0 errors in 30s window
- [ ] Warnings tolerable: GPU-stall ReadPixels (bekend, niet kritiek), maar NIET decode-fail/404/CORS

### 6. Onboarding force-show test
- [ ] Navigate to `?onboard=1` → full sequence (3.5s portal+arrival+walking)
- [ ] Navigate without param + LS-true → 0.6s mini-flash, walking direct
- [ ] localStorage wipe + reload → full sequence

### 7. STOP conditions (any → no deploy)

❌ DNA-incorrect Cosmo (kawaii eyes, vinger-handen, missing antenne, etc.)
❌ NL voice-over hoorbaar
❌ Music drops out na 60s
❌ 4 monden tegelijk op mouth-pillar
❌ Crystal default-zichtbaar
❌ Console errors (niet GPU-stall)
❌ Cosmo doesn't appear / portal doesn't open
❌ FPS < 30 op iPhone-emulation
